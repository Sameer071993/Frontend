### Installation

1. **Clone the repository**

```bash
git clone https://github.com/Sameer071993/Frontend
cd viteproject
```

2. **Install dependencies**

```bash
npm install
# or
yarn install
```

3. **Run the development server**

```bash
npm run dev
# or
yarn dev
```

4. **Open the app**

Visit `http://localhost:5173/frontend/` in your browser.
